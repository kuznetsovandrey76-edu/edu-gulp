Gulp - task runner
Webpack - project builder

Install `https://nodejs.org/en/`  
Первая установка `npm i gulp -g`  

`npm init`
`npm i`  
`gulp build`  
`gulp watch` - use External link    

for Webpack
`npm run dev`  

Структура
```
public_html/
  - index.html
resources/
  - assets/
    - js/
    - sass/
    - img/
      - images/
      - sprites/
```
